Animated Button Effect

A CSS-based animated button with a smooth hover effect that reveals colored text with a glowing border

Hover over the buttons to see the effect

Complete Code

<!-- HTML -->
<button class="button" data-text="Awesome">
    <span class="actual-text"> uiverse </span>
    <span aria-hidden="true" class="hover-text"> uiverse </span>
</button>

<style>
/* === removing default button style ===*/
.button {
  margin: 0;
  height: auto;
  background: transparent;
  padding: 0;
  border: none;
  cursor: pointer;
}

/* button styling */
.button {
  --border-right: 6px;
  --text-stroke-color: rgba(255,255,255,0.6);
  --animation-color: #37FF8B;
  --fs-size: 2em;
  letter-spacing: 3px;
  text-decoration: none;
  font-size: var(--fs-size);
  font-family: "Arial";
  position: relative;
  text-transform: uppercase;
  color: transparent;
  -webkit-text-stroke: 1px var(--text-stroke-color);
}

/* this is the text, when you hover on button */
.hover-text {
  position: absolute;
  box-sizing: border-box;
  content: attr(data-text);
  color: var(--animation-color);
  width: 0%;
  inset: 0;
  border-right: var(--border-right) solid var(--animation-color);
  overflow: hidden;
  transition: 0.5s;
  -webkit-text-stroke: 1px var(--animation-color);
}

/* hover */
.button:hover .hover-text {
  width: 100%;
  filter: drop-shadow(0 0 23px var(--animation-color));
}
</style>

How This Effect Works

This button uses a clever CSS technique to create an animated hover effect:

  • Two Text Layers: The button contains two identical text spans stacked on top of each other.
  • Base Layer: The .actual-text is semi-transparent white with a text-stroke effect.
  • Hover Layer: The .hover-text is colored green (#37FF8B) and initially has 0% width.
  • Hover Animation: When hovering, the .hover-text expands to 100% width, revealing the colored text.
  • Border Effect: A right border creates the typing cursor effect that expands with the text.
  • Glow Effect: The drop-shadow filter creates a glowing effect on hover.

Note: The data-text attribute on the button is referenced in the CSS pseudo-element, though in this implementation the text is hardcoded in the spans. The aria-hidden="true" attribute ensures screen readers don't announce the duplicate text.
